# Meeting Productivity Booster

Instead of being productive, sit and watch a dvd logo bounce around the terminal window. Celebrate when it does. If it 
ever does.

## How to use

```
go build
./dvdScreensaver
```

`ctl-c` or `esc` to exit
`c` to skip to the good part

Because I could.
